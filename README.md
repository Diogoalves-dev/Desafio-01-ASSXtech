## 🚀 Projeto 1 — Integração com WhatsApp para E-commerce

### 📌 Descrição
Este projeto consiste em uma aplicação web que simula um fluxo de atendimento para e-commerce.  
A interface apresenta um botão que redireciona o usuário para o WhatsApp, onde um fluxo automatizado de atendimento foi previamente configurado.

A proposta é demonstrar a integração entre uma aplicação web e canais de comunicação externos, com foco em automação de atendimento e conversão de clientes.

---

### ⚙️ Tecnologias utilizadas

- **Frontend:** React.js  
- **Backend:** Node.js  
- **Integração:** WhatsApp (fluxo automatizado com whatsapp-web.js)  

---

### 💡 Funcionalidades

- Redirecionamento direto para o WhatsApp  
- Integração com fluxo automatizado de atendimento  
- Interface simples e objetiva para conversão de usuários  

---

### 🧠 Conceitos aplicados

- Integração entre frontend e serviços externos  
- Experiência do usuário (UX) voltada para conversão  
- Estruturação de projeto fullstack
